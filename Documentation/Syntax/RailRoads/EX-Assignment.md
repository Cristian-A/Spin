
# Assignment

An **Assignment Expression** is a single instruction
that assigns a value to a variable.

### Syntax:

    + ─╮── [Expression] ──── = ──── [Assignment] ──╭─ +
       ╰────────────── [Short OR] ─────────────────╯

### Declarations used in this Rail:

- [**Expression**](Expression.md)
- [**Short OR**](EX-Short-OR.md)
