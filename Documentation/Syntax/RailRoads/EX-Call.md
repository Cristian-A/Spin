
# Call

A **Call Expression** is a single instruction
that calls a **Function**.

### Syntax:

    + ── [Postfix] ── ( ──╮───────────── > ─────────────╭── ) ── +
                          ╰─╭── > ──[Expression]── > ─╮─╯
                            ╰── < ────── , ─────── < ─╯

### Declarations used in this Rail:

- [**Postfix**](EX-Postfix.md)
- [**Expression**](Expression.md)
