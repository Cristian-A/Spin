
# Do While Statement

Basic example of a `do while` statement.

### Syntax:

    + ───── do ─────── > ── [Statement] ── > ───╮
    ╭────── < ─────────── < ─────────── < ──────╯
    ╰── while ── ( ── [Expression] ── ) ── ; ── +

### Tokens used in this Rail (Regexes):

- **do** - Defined as `do`
- **while** - Defined as `while`
