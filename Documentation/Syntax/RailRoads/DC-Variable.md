
# Variable Declaration

A **Variable Declaration** declares a variable in scope.

### Syntax:

    + ── [Type] ── identifier ─╮─────────────────────╭─ ; ── +
                               ╰─ = ── [Expression] ─╯

### Tokens used in this Rail (Regexes):

- **identifier** - Defined as `[A-Za-z_][A-Za-z0-9_\-]*`

### Declarations used in this Rail:

- [**Type**](DC-Type.md)
- [**Expression**](Expression.md)
