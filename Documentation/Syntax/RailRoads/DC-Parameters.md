
# Parameters

A **Parameter** is a input variable specification.

### Syntax:

    + ── [Type] ── identifier ─╮────────────────── > ────────────────╭─ +
                               |  ╭─────────────── < ─────────────╮  |
                               ╰──╰── , ── [Type] ── identifier ──╯──╯

### Tokens used in this Rail (Regexes):

- **identifier** - Defined as `[A-Za-z_][A-Za-z0-9_]*`

### Declarations used in this Rail:

- [**Type**](DC-Type.md)
