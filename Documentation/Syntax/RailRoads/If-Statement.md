
# If Statement

Basic example of an `if` statement.

### Syntax:

    + ──── if ──── ( ──── [Expression] ──── ) ────╮
    ╭── < ───────────────── < ─────────────── < ──╯
    ╰── [Statement] ──╮────────── > ──────────╭── +
                      ╰─ else ── [Statement] ─╯

### Tokens used in this Rail (Regexes):

- **if** - Defined as `if`
- **else** - Defined as `else`

### Declarations used in this Rail:

- [**Statement**](Statement.md)
