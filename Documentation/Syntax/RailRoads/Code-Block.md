
# Code Block

A **Code Block** consists of curly brackets
surrounding zero or more statements.
It creates a new Context with its own
declarations and memory management.

### Syntax:

    + ── { ─╮────── > ───────╭─ } ── +
            ╰─ [Statements] ─╯

### Declarations used in this Rail:

- [**Statements**](Statements.md)
