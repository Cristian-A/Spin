
# Code Block

A **Code Block** consists of curly brackets
surrounding zero or more declarations.
It creates a new Context with its own
scope and memory management.

### Syntax:

    + ── { ─╮────────────── > ──────────────╭─ } ── +
            |  ╭─────────── < ───────────╮  |
            ╰──╰───── [Declaration] ─────╯──╯
