
# Repeat Until Statement

Basic example of a `repeat until` statement.

### Syntax:

    + ─── repeat ───── > ── [Statement] ── > ───╮
    ╭────── < ─────────── < ─────────── < ──────╯
    ╰── until ── ( ── [Expression] ── ) ── ; ── +

### Tokens used in this Rail (Regexes):

- **repeat** - Defined as `repeat`
- **until** - Defined as `until`

### Declarations used in this Rail:

- [**Statement**](Statement.md)
