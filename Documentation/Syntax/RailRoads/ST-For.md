
# For Statement

Basic example of a `for` statement.

### Syntax:

    + ── for ──── ( ── [Variable Declaration] ───╮
    ╭────────────────────────────────────────────╯
    ╰─── [Expression] ── ; ── [Expression] ── ) ─╮
    ╭────────────────────────────────────────────╯
    ╰─── [Statement] ─── ; ───────────────────── +

### Tokens used in this Rail (Regexes):

- **for** - Defined as `for`
