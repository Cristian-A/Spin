
# Primary

A **Primary Expression** is a single instruction
that represents a **Literal** or evaluates a
**Grouping Expression**.

### Syntax:

    + ─╮── [Literal] ──╭─ +
       ╰── [Grouping] ─╯

### Declarations used in this Rail:

- [**Grouping**](EX-Grouping.md)
- [**Literal**](EX-Literal.md)
