Statements

Spin statements are the building blocks of the language. They can be simple statements, control flow statements, branching statements and iterative statements. We provide a full list of the statements adopted in the language.

Note

The expressions that determine branch and loop conditions must be of Boolean type.

Expression Statement

The expression statement is a statement that will execute the expression and discard its result value. It's composed of an expression immediately followed by a semicolon ;.

Production Rule

expression statement → expression,';';

If Statement

The if statement is a branching statement that will execute code if a condition is met. If the condition is not met and there is an else branch, the instructions in it will be executed instead. It's composed of an if keyword, followed by a conditional expression wrapped in round parenthesis (...) and the statement to execute when the condition is met. It is optionally followed by an else keyword and a statement to execute when the condition is not met instead.

Production Rule

if statement → 'if','(',expression,')',statement,('else',statement)?;

Code examples:

// Check if a number is even or odd:
if (x % 2 == 0) write "even";
else write "odd";
// Faster version:
if (x & 1 == 0) {
    write "faster";
    write "even";
} else {
    write "faster";
    write "odd";
}

While Statement

The while statement will run some statements while a condition is met. If the condition is not met at the start it halts itself. It is composed of a while keyword followed by a condition expression wrapped in round parenthesis (...) and the statement to repeat.

Production Rule

while statement → 'while','(',expression,')',statement;

Code examples:

// print numbers from 1 to 10:
var i = 1;
while (i <= 10) {
    write i;
    i += 1;
}

Until Statement

The until statement will run some statements until a condition is met. If the condition is met at the start it halts itself. It is composed of an until keyword followed by a condition expression wrapped in round parenthesis (...) and the statement to repeat.

Production Rule

until statement → 'until','(',expression,')',statement;

Code examples:

// print numbers from 10 to 1:
var i = 10;
until (i == 0) {
    write i;
    i -= 1;
}

Repeat Statement

The repeat statement will run some statements until or while a condition is met. It will always repeat the statements at least one time. It is composed of a repeat keyword followed by the statement to repeat and the corresponding keyword (while or until) followed by condition expression wrapped in round parenthesis (...).

Production Rule

repeat statement → 'repeat',statement,('while'|'until'),'(',expression,')',';';

Code examples:

Warning

Both code examples will print '0' at least one time, no matter the condition used. 

// print 0 and positive even numbers <= 10;
var p: Integer = 0;
repeat {
    write p;
    p += 2;
} while (p <= 10);
// print 0 and negative even numbers >= -10;
var n: Integer = 0;
repeat {
    write n;
    n -= 2;
} until (n < -10);
